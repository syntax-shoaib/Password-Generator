# 🔐 Random Password Generator

An elegant and responsive password generator app crafted using **HTML, CSS, and JS**. This project generates passwords with a simple click that can easily be copied. This generator app is fully compatible with desktops as well as mobile phones.

---

## 🚀 Features

* 🔑 Create strong random passwords (more than 12 characters long)
* 🔢 Number characters along with uppercase and lowercase alphabets
* 📋 Copy password to clipboard with one click (Mobile support included)
* 📱 Full responsiveness and compatibility
* 🎨 Aesthetic UI with glass effect
* 🎥 Background video for enhanced UI experience

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Media queries)
* JS (DOM manipulation + Clipboard API)

---

## 📌 Working

1. Click the **Generate** button
2. Secure password will be generated randomly
3. Click the copy icon to copy the generated password
4. A success message pops up after copying
